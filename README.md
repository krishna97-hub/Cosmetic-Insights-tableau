# Cosmetic Insights: Navigating Cosmetics Trends and Consumer Insights with Tableau

## Project Overview

**Cosmetic Insights: Navigating Cosmetics Trends and Consumer Insights with Tableau** is a data visualization project developed to analyze cosmetic product data using Tableau. The project helps users explore cosmetic products through interactive charts, KPIs, filters, and visual stories. A responsive Flask web application is used to display the Tableau dashboard online, making it accessible from anywhere.

The project enables users to compare cosmetic products based on brand, category, price, rating, and skin type, helping them make informed purchasing decisions.

---

## Project Objectives

- Analyze cosmetic product data.
- Compare products across different brands and categories.
- Analyze product prices and ratings.
- Identify products suitable for different skin types.
- Build an interactive Tableau dashboard.
- Create a Tableau Story for data-driven insights.
- Integrate the Tableau dashboard into a Flask web application.
- Deploy the application for online access.
